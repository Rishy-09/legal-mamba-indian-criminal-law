# Model Checkpoint

The trained **Legal-Mamba** model checkpoint (`.pth`) is not stored in this GitHub repository because it exceeds GitHub's file size limit (513 MB).

The checkpoint, along with the associated training metrics, is publicly available on Kaggle.
